```
We propose ModelPredictiveSafety, a model-predictive control framework
for ensuring collision avoidance in mobile manipulators operating near
humans. The framework reasons about future trajectories under
uncertainty and adjusts control inputs to stay within a safety
envelope.

The control loop operates as follows. At each timestep, the
manipulator's current state is observed via joint encoders and onboard
depth cameras. A reactive collision-detection module computes the
minimum distance to nearby humans using the current depth frame. If
this distance falls below a learned safety threshold, the control
input is replaced by a deceleration command scaled to the proximity.
Otherwise, the nominal task controller is followed.

We evaluate ModelPredictiveSafety on a 6-DoF arm in a tabletop
scenario with a simulated human reaching across the workspace. Across
200 trials, the system avoided collision in 197 cases. The three
failure cases involved fast human motion (>1.2 m/s) where the safety
envelope was insufficient.

Compared with reactive-only baselines, our approach reduces unnecessary
stops by 23% while maintaining the same collision-avoidance rate. We
argue this demonstrates that model-predictive reasoning yields smoother
human-robot interaction in shared workspaces. An ablation that disabled
the depth-based safety check showed a sharp increase in collision rate,
confirming the role of the safety threshold in determining the
framework's overall behavior across the evaluated trial set.
```
